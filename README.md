# John Wyne A. Tumabiling - Portfolio Website

A modern, responsive portfolio website for a Web & Mobile Developer. This site showcases skills, services, and provides a way to get in touch.

## Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS variables, flexbox, and grid
- **JavaScript**: Vanilla JS for interactivity

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling with responsive breakpoints
- `script.js` - JavaScript functionality (mobile menu, form handling, animations)

## Getting Started

1. Open `index.html` in a web browser
2. The site will display with full responsiveness
3. Test on different screen sizes to see responsive behavior

## Sections

### Hero Section
- Large banner with introduction
- Social media icons
- Call-to-action button

### About Me
- Personal background and expertise
- Skills and technologies list

### Services
- 6 main service offerings with icons

### Contact Section
- Contact form for inquiries
- Social media links
- CV download button (link placeholder)

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    /* ... other colors ... */
}
```

### Content
Update the text and links directly in `index.html`

### Images
Add images to sections as needed
